# jekyll-sitemap-template

Generate your jekyll sitemap without plugin.

## Install

1. Copy `sitemap.xml` inside your jekyll website folder.
2. Run `jekyll build`
3. Your sitemap is ready inside `_site/sitemap.xml`
4.
